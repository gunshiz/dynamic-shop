# dynamic-shop

A dynamic economy shop plugin for Minecraft where stock and prices fluctuate based on daily market events!

## Features
- **Dynamic Restocking:** Every in-game day, a market event automatically restocks items. 
- **Price-Based Scaling:** The chance to restock and the maximum restock amount scale inversely with an item's base price. Highly valuable items will rarely restock and only by a small amount, while common items will replenish quickly!
- **Interactive GUI:** Easy-to-use graphical interfaces for buying and selling items.
- **Sorting & Searching:** Players can sort items by name, price, stock, or search for specific items.

## Commands
- `/shop` - Opens the main shop GUI.
- `/sell` - Opens the sell GUI to sell your items to the shop.
- `/stock` - Displays today's market fluctuations (which items restocked and by how much).
- `/shopadmin` - Manage the shop, add/remove items, set prices, and adjust stock manually.
